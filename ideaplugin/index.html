1.Lombok

2. JRebel and XRebel for IntelliJ

JRebel 可以实时重载代码变更，不让构建、重启、重部署等繁琐的工作中断心流，让开发者可以专注在工作任务；XRebel 则是性能调试工具，在开发 Java 应用时可以实时观察每一段代码对性能产生的影响，并即早找出有潜在问题的代码。
