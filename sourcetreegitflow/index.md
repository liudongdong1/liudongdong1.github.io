# SourceTreeGitFlow


### 1. Sourcetree

> **SourceTree** 是 Windows 和Mac OS X 下免费的 Git 客户端，拥有可视化界面，容易上手操作。同时它也是Mercurial和Subversion版本控制系统工具。支持创建、提交、clone、push、pull 和merge等操作。

- 配置密钥 SourceTree → Tools → Options
