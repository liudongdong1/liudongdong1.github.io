Notes

  • phase value is more sensitive to the limb movement and the RSSI value is more sensitive to the trunk movement.

  • RCS( Radar Cross Section):雷达横截面,衡量雷达对物体的探测程度。较大的RCS表示更容易检测到对象。

    一个物体将有限量的雷达能量反射回源头。影响此的因素包括：

    • 制成目标的材料；
    • 目标物相对于照明雷达信号波长的大小；
    • 目标的绝对大小；
    • 的入射角（在该角度雷达波束击中目标的特定部分，这取决于目标和其定向到雷达源的形状）;
    • 反射角（反射光束离开目标命中部分的角度；取决于入射角）；
    • 发射和接收辐射相对于目标方向的极化。
    • 对于天线，整个RCS可以分为两个单独的组件，即“结构模式RCS”和“天线模式RCS”。RCS的两个组件与天线上发生的两个散射现象有关。当电磁信号落在天线表面时，电磁能的一部分会散射回空间。这称为结构模式散射。能量的剩余部分由于天线效应而被吸收。由于阻抗不匹配，一部分吸收的能量再次被散射回空间，这称为天线模式散射。

  • Fresnel zones: 菲涅耳区间是一系列共聚焦中的一个扁长 椭球发射器和接收器之间和周围的空间区域。发射的无线电波，声波或光波在到达接收器之前可能会遵循略有不同的路径，尤其是在两者之间存在障碍物或反射物体的情况下。波的到达时间可能略有不同，并且由于路径长度不同，它们的相位也会稍有不同。根据相移的大小，这些波会产生相长干涉和相消干涉。在距发射器和接收器任何特定距离处计算出的菲涅耳带的大小可以帮助预测路径上的障碍物或不连续性是否会引起明显的干扰。空间结构：菲涅耳区域是空间中（例如1、2、3）的共焦长 椭圆形区域，以直接传输路径（图中的路径AB）的线为中心。第一区域包括直接视线信号穿过的椭圆空间。如果发射信号的杂散分量从该区域内的物体反弹然后到达接收天线，则相移将小于四分之一长度的波，或小于90º的位移（图中的路径ACB）。仅关于相移的影响将是最小的。因此，该反弹信号可能会对接收器产生积极影响，因为它接收到的信号比没有偏转时要强，并且附加信号可能大部分是同相的。但是，这种偏转的正属性还取决于信号相对于物体的极化（请参见下面有关极化的部分）。

    第二区域包围第一区域，但是不包括第一区域。如果反射物体位于第二区域，则从该物体反弹并被接收器捕获的杂散正弦波将由于路径长度的增加而偏移大于90º但小于270º，并且可能会收到异相。通常这是不利的。但这又取决于极化（下面解释）。

    第三区域围绕第二区域，并且接收器捕获的偏转波将具有与第一区域中的波相同的效果。也就是说，正弦波的偏移量将大于270º但小于450º（理想情况下为360º偏移），因此将以与信号可能从第一个区域到达相同的偏移量到达接收器。从该区域偏转的电波有可能精确地偏移一个波长，因此当它到达接收天线时，它与视线波完全同步。

    第四区域围绕第三区域并且类似于第二区域。等等。

    如果不受阻碍并且处于理想环境中，无线电波将以相对直线的方式从发射器传播到接收器。但是，如果存在与杂散传输波相互作用的反射表面，例如水体，平坦的地形，屋顶，建筑物的侧面等，则从这些表面偏转的无线电波可能异相或进入与直接传输到接收器的信号相位。有时，这会导致违反直觉的发现，即降低天线的高度会增加接收器的信噪比。

    尽管无线电波通常沿直线传播，但雾气甚至湿度会导致某些频率的某些信号在到达接收器之前发生散射或弯曲。这意味着远离视线路径的物体仍可能会阻塞信号的一部分。为了最大程度地提高信号强度，需要通过从直接射频视线（RF LoS）以及主要菲涅耳区域内的周围区域去除障碍物来最大程度地减少阻塞损失的影响。最强的信号在发送器和接收器之间的直线上，并且始终位于第一个菲涅耳区域。

    在19世纪初期，法国科学家Augustin-Jean Fresnel发现了一种计算区域位置的方法-也就是说，给定的障碍物将导致发射器和接收器之间的同相偏转还是异相偏转。

Paper《RF-3DScan: RFID-based 3D Reconstruction on Tagged Packages》

Phenomenon&Challenge:

  1. uncertain tag direction is easy to create dead zones of RFID communication.,how to avoid dead zones and achieving the robust 3D recconstruction
  2. existing work can only derive the 2D relative localization of tag objects via once mobile scanning. build an angle-profile model and combine this model with the priori knowledge of tag layout to sense the package placement in 3D space
  3. RF signal is likely to be distorted due to the tag orientation or the multi-path effect,how to select effective data to ensure the performance phase differences to derive angle profiles to eliminate the phase variation caused by the changing tag’s orentation relative to the antenna
  4. propose an adaptive algorithm to automatically filter outliers and keep remaining data for the later estimation

  1. due to the frequent rearrangement of tagged-ogjects ,a fast RFID based tracking approach is highly desired for accurate logistics distribution
