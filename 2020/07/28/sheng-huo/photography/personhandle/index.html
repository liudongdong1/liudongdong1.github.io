1. 皮肤通透性

  • 选取颜色： 黄色，红色 减青减红。
  • 添加曲线，ctrl+shift+out+2 选择高光区域，然后反选，提高亮度（亮白）。

2. 污点去除

[] 是用来调节大小的，替换的圆圈内容可以选择。

3. 虚化

  • 前景虚化

  • 背景虚化

  • 几何（散景）虚化

拍摄场景中存在（点状）光源，这些（点状）光源模糊后就会形成几何形状的光斑，形成浪漫梦幻的效果。

  1. 大光圈，中焦镜头上的长焦端。
  2. 拍摄与背景距离远的主体：**拍摄花卉散景照片时，合理选择拍摄角度，选择花朵与背景较远的角度进行拍摄可以让你的背景变得更加模糊。
  3. 凑近主体拍摄缩小拍摄距离：
  4. 确认光源方向选择顺光/逆光拍摄：顺光拍摄的主体会更明亮，背景更加柔和。逆光拍摄：主体的背景中就会有许多（点状）光，你就可以拍摄出浪漫的光斑效果。

3.1. 虚化后期处理

ctrl+j；复制图层；滤镜–模糊–镜头模糊； 然后添加蒙版，将主体部分擦拭出来。

3.2. 羽化操作

  • 选择一个矩形框，然后羽化
  • 选择—修改—羽化

4. 测光

测光区域不要对准纯白或者纯黑的画面，要对着有颜色不深不浅的物体进行测光！对于纯白，纯黑的使用 白加黑减策略。在半自动档模式下测光才起作用,

点测光：是对拍摄视野中央极小的区域进行测光计算平均值，这一点大约在取景器中央部位大约3.8%的区域，而这个点通常和对焦点处于同一位置。使用场景：日出日落、烟花、逆光拍摄、剪影、特写、人像拍摄等

评价测光：是对整个拍摄视野画面进行测光计算平均值，先将取景器中的画面切割成多个区域，然后每个被割成独立的区域进行测光取平均值，最后通过各种计算得出一个相机认为准确的曝光值。

局部测光：是对拍摄视野中央小圆画面进行测光计算平均值，相机的测光元件会对画面中心处约占画面12%的范围进行测光。

中央重点平均测光一般在主体占画面较大面积，是对视野中央重点测光兼顾整体，或者在光比与色彩差异比较大时使用。相机的测光会把画面中央约60%的区域作为测光的重点进行测光。适合纪实作品、人物肖像、旅游拍照等。

4.1. 逆光剪影

  • 曝光与测光： 拍摄逆光应该将测光模式设置为点测光，以被摄主体脸部作为曝光基准。测光点应放在离拍摄主体不远的较亮的地方测光（单反摄影控：smmapp），拍摄主体的轮廓剪影会清晰一些。宁可曝光不足也不可过曝。如果曝光过度，剪影的黑色部分不够黑，背景也会过曝无细节。拍摄时也可以用灯串为画面添加光斑，让氛围更浓。降低曝光补偿 天空色彩迷人，人物只有轮廓 阴影白平衡

  • 背景与角度选择

    逆光拍摄的背景多数是天空，但是尽量不要选择大片天空。因为大片天空背景很亮，如果让被摄主体脸部正确曝光，天空就是一片死白，没有任何细节，过曝了。

    选择一些相对比较暗的背景，或者将颜色较深物体的位置作为背景，例如小树林或灰色建筑等，这样就可以避免背景过曝无细节或者主体太黑的尴尬情况。

  • 拍摄时间：日落前1小时是拍逆光人像的最佳时间，这时段的光线呈温暖的金黄色，最柔和，且光亮反差比较小，太阳位置比较低，色温也比较低，可以拍出环境的层次感和主体的立体感。

  • 参数设置：ISO设置尽可能低，例如ISO100。尽量使用大光圈镜头进行拍摄，例如：F1.2或F1.4。因为大光圈能让背景分离，主体更突出，且让你拍摄逆光照时无需反光板。

    采用光圈优先，一般用小光圈（例如：F10和F12）。ISO设置为100最好。如果在海边拍摄剪影，为了把海水拍出镜面的效果，可以将相机固定在三脚架上，采用慢速快门拍摄。

  • 建在海滩上的展台或者白色岩石，亦或者白色或银色的车，这些擅长反射光线的物体都可以作为反光板。例如：将被摄主体安排在岩石附近，太阳照射在岩石上，岩石上的光再反射到被摄主体的脸上，就相当于给被摄主体的脸补光了。

先半按快门自动对焦，对焦后不松手，把镜头上的自动对焦（AF）改成手动对焦（MF），并锁定曝光，这样对焦就不是问题了（单反摄影控：smmapp）。等被摄主体进入画面理想位置，高速连拍，总有一张是满意的。

  • 拍摄角度：（蹲下拍，再不行就趴着！）来仰拍或平拍 角度低/天空面积大/轮廓清晰 衣服贴身/轮廓好看 人物轮廓不要重叠 注意腿部轮廓，尽量不要用双腿并拢的姿势 拍人物特写尽量拍侧面

6. 海景

日出的海面、日落的海面、栈桥、灯塔、房子、海滩、轮船、海鸟、海边物产丰富，早晨退潮后没来得及跑掉的小螃蟹、小海龟、漂亮的海星，都可以用超广角拍出非常震撼、漂亮的照片。除此之外，海边的房子、遮阳伞、栈桥、海滩、海湾、海岛、海鸟都可以作为拍摄的主体。

  • 留意水平线，不能让海天相间的水平线拍斜。
  • 加入引导线，比如栈桥，比如海岸线。
  • 使用不同焦距，多景别多拍几张喜欢的场景，不留遗憾。
  • 避免使用广角镜头时把垂直的物体放在照片的边缘，容易因为畸变而变形。
  • 给你的照片加入对比，比如高大的灯塔与岸边的小小人物的大小对比；比如夜色下沙滩上帐篷里灯光的冷暖对比等等。
  • 学会留白。海阔天空，一切都显得渺小，不妨留白。
    • 慢门曝光

7. 前景虚化

框架式前景 ： 利用好周边的各种框架，如门与窗、两物之间的夹缝或是自制的“框架”……利用这些框架，对焦被摄主体，可以打造出一种视觉的窥视感，集中人们的视线，突出画面主体。

引导式前景：存在一些具有一定延伸性的前景物（如道路、栏杆、扶梯、墙面等等），可以尝试将被摄人物安排在这种类型前景物的延伸处。拍摄时，将镜头往前景物的一边靠近，与被摄人物保持一定的距离，同时保留画面的延伸趋势，这样观者的关注焦点便会集中在视觉引导线上的人物主体上。

虚化式前景： 而所谓的虚化式前景，是当镜头近距离贴近前景物时，彻底虚化前景，形成一团拥有一定色调的虚影，随后将对焦点放置在人物身上，即可获得一张虚化式前景的人像照。
